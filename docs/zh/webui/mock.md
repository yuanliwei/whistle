1. URL Pattern：匹配请求 URL 的表达式，参见：(pattern)[../pattern.html]
2. Operation：操作指令
   -  操作协议：(协议列表)[../rules]
   -  操作值类型：Inline 或 Key-Value 模式
   -  Inline：内嵌到 Rules 里面
   -  Key-Value：设置 Key Name，选择默认值，编辑内容，最终存到 Values 里面
   -  Rules：显示当前将生成的规则（可以删除或 Copy）
